# Elite Oracle V2

**面向创始人、投资人、运营者、研究员和建设者的多语言深度研究与精英战略分析 Skill。**

Elite Oracle V2 可以把分散的信息转化为可用于决策的洞察。它将顶级战略咨询公司、风险投资机构、分析机构、投行、资管机构、宏观研究机构和管理思想家的公开输出模式，整合成一个实用的分析操作系统。

它不冒充任何真实机构，也不声称拥有专有方法或内部资料。它只基于公开模式、用户提供的材料和可引用研究进行分析。

## V2 升级点

V2 将 Elite Oracle 从战略综合 Skill 升级为更强的研究和信号智能系统。

新增能力：

- 在相关场景下横跨英语、日语、中文进行研究
- 优先使用官方来源和一手资料
- 支持 GitHub、技术文档、财报、X、Reddit、论坛和一线从业者信号
- 在分析前进行证据聚合
- 构建 actor map、claim map、issue tree、timeline、contradiction list
- 以顶级 VC 视角识别深层趋势
- 发现隐藏瓶颈和价值控制点
- 输出带风险、KPI 和失败信号的执行建议

## 它回答什么问题

Elite Oracle 适合回答：

1. 什么正在变化？
2. 为什么是现在？
3. 表层趋势背后的深层驱动是什么？
4. 谁会赢，谁会输，谁会掌握控制点？
5. 哪些证据支持这个判断？
6. 哪些证据可能推翻它？
7. 决策者下一步应该做什么？
8. 什么信号会让我们改变观点？

## 研究来源

Elite Oracle V2 可以综合：

- 公司官网、官方博客、产品文档
- 财报、年报、投资者材料、业绩材料
- 政府、监管、标准组织、法院或执法材料
- GitHub、changelog、issue、RFC、model card、论文
- 咨询报告、VC 文章、分析师资料、投行公开研究
- 可信商业、科技、金融和行业媒体
- X、Reddit、Hacker News、专业论坛和专家评论

## 安装

将 `skill/` 文件夹复制到：

```text
~/.codex/skills/elite-oracle
```

重启 Codex 后调用：

```text
$elite-oracle
```

## 快速开始

```text
$elite-oracle

Analyze whether vertical AI agents will become a new enterprise software category.
Use official sources, VC essays, market reports, GitHub signals, and operator commentary.
Produce a VC thesis with hidden bottlenecks, control points, risks, and experiments.
```

## 安全和诚信

Elite Oracle：

- 不冒充 McKinsey、BCG、Bain、a16z、Sequoia、Goldman Sachs、Gartner 等真实机构
- 不声称拥有内部资料或专有方法
- 标注假设、不确定性和推理
- 只使用公开资料和用户提供材料
- 优先追求可用于决策的分析，而不是好看的空话
