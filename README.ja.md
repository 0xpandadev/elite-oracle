# Elite Oracle V2

**多言語ディープリサーチ、シグナル集約、戦略コンサル/VC/投資銀行レベルの分析を行うエージェントスキルです。**

Elite Oracle V2 は、散らばった情報を意思決定に使える洞察へ変換するための再利用可能なスキルです。戦略コンサル、トップVC、調査会社、投資銀行、資産運用会社、マクロリサーチ機関、経営研究の公開アウトプットに見られる思考パターンを、実務で使える分析OSとして統合しています。

特定企業のなりすましではありません。非公開の手法や内部資料へのアクセスを主張しません。公開情報から抽出できる分析パターンを、実務に使える形へ再構成したものです。

## V2 の強化点

V2 では、従来の「戦略的な示唆を出すスキル」から、より強力なリサーチ/シグナルインテリジェンス型へ進化しています。

追加された能力:

- 英語・日本語・中国語を必要に応じて横断する多言語調査
- 公式情報・一次情報を起点にした調査
- GitHub、技術ドキュメント、有価証券報告書、X、Reddit、フォーラム、オペレーター発信の活用
- 分析前の情報集約
- アクターマップ、クレームマップ、論点ツリー、タイムライン、矛盾リストの作成
- トップVC的な深層トレンド検知
- 隠れたボトルネックや制御点の発見
- リスク、KPI、失敗シグナルを含む経営向け提言

## 何ができるか

Elite Oracle は次の問いに答えるために使います。

1. 何が変化しているのか
2. なぜ今なのか
3. 表面的なトレンドの裏にある深いドライバーは何か
4. 誰が勝ち、誰が負け、誰が制御点を握るのか
5. どの証拠がそれを支えているのか
6. 何が反証になり得るのか
7. 意思決定者は次に何をすべきか
8. 何が起きたら見方を変えるべきか

## 調査対象ソース

Elite Oracle V2 は以下を横断して考えます。

- 企業公式サイト、公式ブログ、製品ドキュメント
- 有価証券報告書、年次報告書、投資家向け資料、決算資料
- 政府、規制当局、標準化団体、裁判・行政資料
- GitHub、changelog、issue、RFC、model card、論文
- コンサルレポート、VCエッセイ、アナリスト資料、投資銀行の公開リサーチ
- 信頼できるビジネス・テック・金融・業界メディア
- X、Reddit、Hacker News、専門フォーラム、実務家コメント

## インストール

`skill/` フォルダを以下に配置してください。

```text
~/.codex/skills/elite-oracle
```

Codex を再起動して、次のように呼び出します。

```text
$elite-oracle
```

## クイックスタート

```text
$elite-oracle

垂直型AIエージェントが新しいエンタープライズソフトウェアカテゴリになるかを分析して。
公式情報、VCエッセイ、市場レポート、GitHubシグナル、実務家コメントを見て、
隠れたボトルネック、制御点、リスク、検証実験を含むVC thesisとして整理して。
```

## 安全性と誠実性

Elite Oracle は以下を守ります。

- McKinsey、BCG、Bain、a16z、Sequoia、Goldman Sachs、Gartner などの実在機関になりすまさない
- 非公開手法や独自データへのアクセスを主張しない
- 仮定、不確実性、推論を明示する
- 公開情報とユーザー提供情報のみを使う
- かっこよい文章より意思決定に使える分析を優先する
