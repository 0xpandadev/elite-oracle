# Elite Oracle 日本語説明

**Elite Oracle（エリートオラクル）** は、世界トップクラスの知的集団が公開しているアウトプットの「型」を蒸留した、再利用可能な分析スキルです。

対象にした知的集団は、戦略コンサル、トップVC、調査会社、投資銀行、資産運用会社、マクロリサーチ機関、経営学・経済学系の研究機関です。特定企業を真似るものではなく、公開情報から抽出できる汎用的な分析習慣を、一つの思考OSとして再構成しています。

## 何に使えるか

- 市場調査
- 競争環境分析
- AI・技術トレンド分析
- スタートアップの事業仮説
- VC投資仮説
- 投資メモ
- 経営向けブリーフ
- コンサルティング資料のストーリーライン
- 業界構造分析
- M&A・資本市場・マクロ環境の整理

## 中核の問い

Elite Oracle は、どんなテーマでも次の7問に戻して考えます。

1. 何が変わっているのか。
2. なぜ今なのか。
3. 誰が勝ち、誰が負けるのか。
4. その根拠は何か。
5. 隠れた制約は何か。
6. 次に何をすべきか。
7. 何が起きたら、この見立てを変えるべきか。

## 蒸留した思考パターン

- **McKinsey / MGI型:** 大きな不確実性を経営者が意思決定できる論点に分解する。
- **BCG / Henderson Institute型:** 競争優位、戦略仮説、組織能力、シナリオで未来を読む。
- **Bain型:** 顧客経済、利益プール、実行KPI、PE的な実務性に落とす。
- **Big Four / Accenture型:** 大企業の変革、ガバナンス、規制、リスク、導入現実を入れる。
- **a16z / Sequoia / YC型:** 技術波、創業仮説、初期市場、ウェッジ、ディストリビューションを見る。
- **Gartner / Forrester / IDC型:** カテゴリ分類、成熟度曲線、導入タイミング、ベンダー比較を作る。
- **Goldman / JPMorgan / Morgan Stanley型:** マクロ、資本フロー、バリュエーション、M&A、収益ドライバーに接続する。

## 12レンズ分析マトリクス

1. 市場構造: 規模、成長、集中度、利益プール、ボトルネック。
2. 顧客現実: 痛み、予算所有者、スイッチングコスト、導入摩擦。
3. 技術成熟度: 性能曲線、コスト曲線、信頼性、相互運用性。
4. ビジネスモデル: ユニットエコノミクス、収益化、粗利、継続性。
5. 競争優位: データ、流通、採用力、ブランド、規制、ワークフロー固定化。
6. 資本市場: 資金調達、バリュエーション、M&A、出口環境。
7. オペレーティングモデル: 組織、プロセス、インセンティブ、ガバナンス。
8. リスクと規制: コンプライアンス、セキュリティ、評判、地政学。
9. タイミング: 成熟曲線、ハイプサイクル、S字カーブ、トリガー。
10. シナリオ: ベース、アップサイド、ダウンサイド、非連続変化。
11. 指標: 先行指標、遅行指標、閾値、失敗シグナル。
12. ナラティブ: なぜ今これが重要かを一言で説明する物語。

## 使い方

Codexで使う場合は、`skill/SKILL.md` を以下へ配置します。

```text
~/.codex/skills/elite-oracle/SKILL.md
```

その後、次のように呼び出します。

```text
エリートオラクルで、AIエージェントがエンタープライズSaaS市場をどう変えるか分析して。
```

または、`colleague/work.md` と `colleague/persona.md` を任意のAIに渡して、同じ分析スタイルを再現できます。

## 出力モード

- **Executive Brief:** 経営者向けの意思決定メモ。
- **VC Thesis:** スタートアップ・技術波・市場創造の投資仮説。
- **Consulting Deck Narrative:** コンサル資料のストーリーライン。
- **Analyst Landscape:** 市場、技術、ベンダー、カテゴリの俯瞰。
- **Investment / Macro Memo:** 投資、M&A、バリュエーション、マクロ環境の分析。

## 注意

このプロジェクトは、特定企業の非公開ノウハウや機密情報を使っていません。公開アウトプットに見られる汎用的な構造、問いの立て方、分析の順序、品質基準を抽出したものです。
