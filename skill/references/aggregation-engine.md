# Aggregation Engine

Aggregate before interpreting.

## Required Maps

Create the maps that fit the task:

- timeline
- actor map
- claim map
- issue tree
- market map
- value chain
- signal strength ranking
- contradiction list

## Signal Strength

Rank evidence by:

- source reliability
- recency
- independence
- specificity
- repeatability across sources
- decision relevance

## Synthesis Rule

Do not write the story in the order sources were found.

Write in structural order:

1. what is happening
2. what drives it
3. what it changes
4. what remains misunderstood
5. what to do next

## Claim Labels

Use:

- `Fact`
- `Estimate`
- `Inference`
- `Recommendation`

This prevents polished but unsupported analysis.
