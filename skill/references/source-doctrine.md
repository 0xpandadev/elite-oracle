# Source Doctrine

Elite Oracle V2 searches broadly, then ranks rigorously.

## Source Universe

Use these buckets when relevant:

1. primary official sources
2. technical and documentary sources
3. elite public thought leadership
4. high-credibility media and research
5. operator and community signal

## Primary Official Sources

- company sites
- official blogs
- product docs
- filings
- earnings materials
- government and regulatory pages
- standards bodies
- court or enforcement materials

## Technical And Documentary Sources

- GitHub
- changelogs
- issues
- RFCs
- papers
- model cards
- engineering posts
- API documentation

## Elite Public Thought Leadership

- consulting reports
- VC essays
- analyst notes
- investment bank public research
- macro letters
- management research

## Operator And Community Signal

- X
- Reddit
- Hacker News
- niche forums
- expert comment threads

Use these to detect implementation friction, adoption reality, skepticism, and weak signals.

## Multilingual Rule

If the topic touches global technology, Japan, China, the US, or cross-border markets, search beyond one language.

Consider:

- English
- Japanese
- Chinese

## Ranking Rule

Primary sources anchor facts.

Secondary sources add context.

Community sources reveal friction, narrative, and weak signals.

Do not collapse them into one confidence level.
