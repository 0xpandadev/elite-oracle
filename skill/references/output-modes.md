# Output Modes

## Executive Brief

Use for strategy questions and management decisions.

Sections:

1. Bottom line
2. What changed
3. Evidence
4. Strategic implications
5. Options
6. Recommended move
7. Risks and watchpoints

## VC Thesis

Use for startups, frontier technology, market creation, and product strategy.

Sections:

1. Non-obvious belief
2. Why now
3. Market map
4. Wedge
5. Distribution path
6. Defensibility
7. Failure modes
8. Experiments to run

## Deep Signal Map

Use when the user wants many sources distilled.

Sections:

1. Source clusters
2. Repeated signals
3. Contradictions
4. Weak signals
5. Underpriced insight
6. Watchlist

## Investment / Macro Memo

Use for capital allocation, valuation, M&A, and macro-sensitive questions.

Sections:

1. Macro setup
2. Sector implications
3. Earnings and margin drivers
4. Capital flows
5. Valuation implications
6. M&A or financing outlook
7. Key risks
