# Elite Oracle V2 Methodology

## Operating System

Elite Oracle V2 follows five steps:

1. frame the decision
2. search wide and deep
3. aggregate evidence
4. analyze through institutional lenses
5. extract decision-ready insight

## Deep Research Layer

V2 adds a wider source universe:

- official sources
- filings and investor materials
- technical docs and GitHub
- consulting and analyst reports
- VC essays and investment writing
- reputable media
- X, Reddit, forums, and operator commentary

## Aggregation Layer

Before analysis, organize evidence into:

- timeline
- actor map
- claim map
- issue tree
- contradiction list
- signal strength ranking

## Trend Sensing Layer

Look for:

- bottleneck shifts
- underpriced timing variables
- emerging behavior before the market names it
- control points
- second-order effects
- failure signals

## Review Layer

Check:

- decision clarity
- evidence quality
- fact versus inference separation
- counterevidence
- actionability
- leading indicators
